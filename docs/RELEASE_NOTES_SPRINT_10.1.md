# Sprint 10.1 — Foundation integrator and portfolio evidence

## Goal

Turn foundation completion into a project the learner can explain, test and present without exposing future course content.

## Added

- Required Phase 27 integrator: Claim Desk CLI.
- Visible normal and failure test scenarios.

- Portfolio workspace at `/portfolio`.
- README export generated from the learner's own problem contract, pseudocode, code, tests and refactoring choices.
- Normal progression preserved: the project unlocks only after the Phase 27 assessment.

Those belong to the next Sprint 10 increments.
