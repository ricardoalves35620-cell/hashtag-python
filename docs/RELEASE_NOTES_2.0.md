# Hashtag Python 2.0 — Python Profissional

## Publicado

- Fases 28–39 com projetos, ambientes virtuais, dependências, módulos, pacotes, CLI, Git, pytest, debugging, logging, tipos, dataclasses, OOP e capstone profissional.
- Laboratório de projeto com oito missões práticas de terminal e Git.
- Catálogo de fases por módulo para evitar uma lista única excessivamente longa.
- Metadados de trilha, estágio, duração e necessidade de prática desktop.
- Novas habilidades profissionais conectadas ao motor adaptativo.

## Limites honestos

O navegador ensina e avalia conceitos executáveis em Python. Comandos de sistema, Git e ambientes virtuais são ensaiados no laboratório, mas a prática final deve ser repetida no computador real.
