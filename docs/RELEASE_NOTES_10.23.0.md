# Hashtag Python 10.23.0

## V11 curriculum migration: phases 17–20

This release rewrites four boundary-focused phases to the authored V11 gold standard:

- Phase 17 — Reading Files
- Phase 18 — Writing Files
- Phase 19 — JSON Data
- Phase 20 — Date and Time

Each phase now contains a bilingual authored lesson, four progressive exercises, exact normalized grading, AST requirements, public and hidden cases, a varied quiz, and a generalization exam. Exercises use the Pyodide virtual file system for real file operations and include empty, accented, negative, malformed, leap-day and large-value cases.

The V11 migration registry and quality baseline now protect phases 9–20 from content, exercise, hidden-test, exact-grading and AST regressions.
