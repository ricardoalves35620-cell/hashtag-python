# Foundation Sprint 1.1 — Design Tokens and Themes

## Changed

- Replaced the old color palette with semantic light and dark theme tokens.
- Added typography, spacing, radius, elevation, motion, and safe-area scales.
- Added semantic Tailwind utilities while retaining temporary compatibility aliases.
- Improved global text contrast, focus rings, scrollbars, code surfaces, and lesson prose.
- Added reduced-motion support.
- Prevented theme flash during startup.
- Synchronized the browser/PWA theme color with the active theme.
- Added `docs/DESIGN_SYSTEM.md` as the visual source of truth.

## Not included yet

Component redesign begins in Foundation Sprint 1.2. Existing screens already inherit the new palette and typography, but individual buttons, cards, inputs, and alerts will be standardized in the next delivery.
